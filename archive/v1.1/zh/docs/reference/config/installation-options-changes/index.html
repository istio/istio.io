    用 Helm 进行 Istio 的定制安装过程中，release-1.1 和 release-1.0 的部分选项是有差异的，下表描述了这些差异。表格内容分了三类：

    • 在原版本中已经包含了这一安装选项，但在新版本中其取值或描述发生了变化。
    • 新版本中加入了新的安装选项。
    • 新版本中移除了原有安装选项。
