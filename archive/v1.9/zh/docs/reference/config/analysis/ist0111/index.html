    MultipleSidecarsWithoutWorkloadSelectors

    消息名称MultipleSidecarsWithoutWorkloadSelectors
    消息代码IST0111
    描述More than one sidecar resource in a namespace has no workload selector
    等级Error

    This message occurs when more than one Sidecar resource in a namespace does not define any workload selector. This can lead to undefined behavior. See the reference for the Sidecar resource for more information.

    To fix this, ensure that each namespace has only one Sidecar resource without a workload selector.

    MTLSPolicyConflict
