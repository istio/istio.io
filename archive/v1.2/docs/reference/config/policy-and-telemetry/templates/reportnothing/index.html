Report Nothing

  1. Template

The reportnothing template represents an empty block of data, which can be useful in different testing scenarios.

Example config:

apiVersion: "config.istio.io/v1alpha2"
kind: instance
metadata:
  name: reportrequest
  namespace: istio-system
spec:
  compiledTemplate: reportnothing

Template

ReportNothing represents an empty block of data that is used for Report-capable adapters which don’t require any parameters. This is primarily intended for testing scenarios.
