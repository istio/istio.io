参与社区

  1. 相关内容

Istio 是一个开源项目，拥有一个活跃的社区支持它的使用和持续开发。我们非常欢迎您融入其中! 融入 Istio 社区有很多种方式：

前往 Istio 讨论板 提出问题、参与讨论或者寻求帮助解决问题。您可以使用 GitHub ID （或者 Google、Twitter 或 Yahoo）登录！
前往 Stack Overflow 获取关于部署、配置和使用 Istio 实际问题的解答。
如果您想与我们社区的成员进行实时互动，您可以加入 Istio 的 Slack 工作区。
您可以关注 社区日历 参与我们的社区活动.
关注我们的 Twitter 获取最新消息。
GitHub 是进行 Istio 代码开发的地方。
如果您想深入了解 Istio 的具体细节，请查看我们不断丰富的设计文档集。如需要访问这些文档，只需加入 istio-team-drive-access@ 讨论组。 加入之后，您就可以直接前往 文件夹（这里有很多技术文档） 查看。
如果您想为 Istio 项目作出贡献, 请阅读我们的 行为规范 和 贡献指南, 并考虑参与我们的 工作小组。
如果您想了解更多关于 Istio 社区治理的信息，请了解我们作为一个开源项目是如何做事的， 并阅读 Istio 指导委员会和技术监督委员会的详细信息。
中文内容由 ServiceMesher community 维护，加入我们并参与进来吧！
