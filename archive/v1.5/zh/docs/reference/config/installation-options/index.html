使用 Helm 安装 Istio 的方式正在被弃用，不过你在使用 istioctl 安装 Istio 时仍然可以使用这些 Helm 的配置项，把 values. 作为选项名的前缀。例如，替换下面的 helm 命令：

$ helm template ... --set global.mtls.enabled=true

可以使用 istioctl 命令：

$ istioctl manifest generate ... --set values.global.mtls.enabled=true

参考自定义配置获取详细信息。

不幸的是，由于支持的选项集有最新的变化，此文档已经过时。获取准确的支持的选项集，请参阅 Helm charts。
