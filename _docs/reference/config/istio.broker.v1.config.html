CatalogEntry

CatalogEntry defines listing information for this service within the exposed catalog. The message is a subset of OSBI service fields defined in https://github.com/openservicebrokerapi

Field Type Description
name string

Required. Public service name.
id string

Required. Public unique service guid.
description string

Required. Public short service description.

CatalogPlan

CatalogPlan defines listing information for this service plan within the exposed catalog. The message is a subset of OSBI plan fields defined in https://github.com/openservicebrokerapi

Field Type Description
name string

Required. Public service plan name.
id string

Required. Public unique service plan guid.
description string

Required. Public short service plan description.

Deployment

Deployment defines how the service instances are deployed.

Field Type Description
instance string

For truely multi-tenant service, the deployed service instance name.

ServiceClass

ServiceClass defines a service that are exposed to Istio service consumers. The service is linked into one or more ServicePlan.

Field Type Description
deployment Deployment

Required. Istio deployment spec for the service class.
entry CatalogEntry

Required. Listing information for the public catalog.

ServicePlan

ServicePlan defines the type of services available to Istio service consumers. One or more services are included in a plan. The plan is flexible and subject to change along with business requirements.

Field Type Description
plan CatalogPlan

Required. Public plan information.
services string[]

Required. List of the Keys of serviceclass config instance that are included in the plan. ServiceClass is a type of CRD resource.
