This package defines common string protos.

StringMatch

Describes how to match a given string. Match is case-sensitive.

Field Type Description
exact string (oneof)

exact string match.
prefix string (oneof)

prefix-based match.
suffix string (oneof)

suffix-based match.
regex string (oneof)

ECMAscript style regex-based match as defined by EDCA-262. Example: “^/pets/(.*?)?”
