Istio Annotations that can be applied to other Mesh resources.

Sidecar

Sidecar annotations alter the behavior of sidecars.

See Automatic Sidecar Injection
Annotation Type Description
sidecar.istio.io/inject boolean

The sidecar injector will inject the sidecar into pods by default. Add the sidecar.istio.io/inject annotation with value false to the pod template spec to override the default and disable injection.

policy.istio.io/check string

Workload-bound annotations for controlling mixerfilter
policy.istio.io/checkRetries number

Workload-bound annotation for max number of retries on transport error
policy.istio.io/checkBaseRetryWaitTime string in duration format

Workload-bound annotation for base time to wait between retries, will be adjusted by backoff and jitter. Example: 80ms.
policy.istio.io/checkMaxRetryWaitTime string in duration format

Workload-bound annotation for max time to wait between retries. Example: 1000ms.
