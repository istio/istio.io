서비스를 연결, 보호, 제어 및 관찰해보세요.

{{< inline_image "landing/routing-and-load-balancing.svg" >}}
연결

서비스 간 트래픽 및 API 호출 흐름을 지능적으로 제어하고, 다양한 테스트를 수행하며, red/black 배포를 통해 점진적으로 업그레이드합니다.

{{< inline_image "landing/resiliency.svg" >}}
보호

인증, 인가 및 서비스 간 통신 암호화를 통해 자동으로 여러분의 서비스를 보호합니다.

{{< inline_image "landing/policy-enforcement.svg" >}}
제어

정책을 적용하여, 정책이 잘 시행되고 리소스가 소비자에게 공평하게 분배되는지 확인합니다.

{{< inline_image "landing/telemetry-and-reporting.svg" >}}
관측

여러분의 모든 서비스에 풍부하고 자동화된 트레이싱, 모니터링 및 로깅을 더하면 무슨 일이 벌어지는지 알아봅니다.
