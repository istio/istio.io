Un gateway es un proxy Istio independiente desplegado en el borde de la malla.
Los gateways se usan para enrutar tráfico [hacia](/es/docs/tasks/traffic-management/ingress/) o [fuera de](/es/docs/tasks/traffic-management/egress/) la malla.

Un CR `Gateway` de Istio se usa para configurar los puertos expuestos de un deployment de gateway.
