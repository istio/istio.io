A sidecar, generally, is a container that runs alongside a primary application to provide additional functionality.
In Istio, sidecar mode is a [data plane mode](/es/docs/reference/glossary/#data-plane-mode) that runs an [Envoy](/es/docs/reference/glossary/#envoy) proxy alongside each
[Pod](/es/docs/reference/glossary/#pod).
