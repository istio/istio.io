Welcome to the Istio documentation contribution guides. This section contains
all the information you need to perform the following documentation tasks:

- Make a [quick edit](/about/contribute/github/#quick-edit) to existing documentation
- [Add new documentation](/about/contribute/add-content)
- [Review documentation contributions](/about/contribute/review)
- [Build a local version of the website](/about/contribute/build) to preview and
  test your documentation contributions
- Include the required [front matter](/about/contribute/front-matter)
- Include [code blocks](/about/contribute/code-blocks) in your content
- Include [consistent diagrams](/about/contribute/diagrams)
- [Format](/about/contribute/formatting) your content consistently
- Use the [implemented shortcodes](/about/contribute/shortcodes) for tabs,
  callouts and other forms of dynamic content
- Follow our [style guidelines](/about/contribute/style-guide)
- Use the proper [terminology](/about/contribute/terminology)

The Istio documentation is multilingual. Within the `content` folder, you can
find the following folders for the available languages:

- The English original content is in the `en` folder.
- The Mandarin translation is in the `zh` folder.
