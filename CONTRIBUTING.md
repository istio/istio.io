# Contribution guidelines

So you want to hack on Istio? Yay! Please refer to Istio's overall
[contribution guidelines](https://github.com/istio/community/blob/master/CONTRIBUTING.md)
to find out how you can help.

## Localized translations

If you are translating or editing any localized content, follow the translation conventions for that locale:

- `I18N-AGENTS-LOCALE.md` (for example, `I18N-AGENTS-ES.md`)
