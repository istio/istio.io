 我们的社区

加入社区

  • See also

Istio 是一个开源项目，拥有一个支持其使用和持续开发的活跃社区。我们很乐意您加入我们并参与其中！

融入 Istio 社区有很多种方式：

加入 istio-users@ 邮件列表以参与讨论和获取帮助。 加入 istio-announce@ 邮件列表以了解最新版本、活动和其他事件。 还有一个 istio-dev@ 邮件列表用于讨论围绕 Istio 项目开发的问题。
关注我们的 Twitter 获取最新消息。
Stack Overflow 上有关于部署、配置和使用 Istio 等实际问题的解答。
在 Rocket Chat 上与 Istio 社区成员交流。
GitHub 用于托管 Istio 的代码并在其上开发。
如果您想深入了解 Istio 的具体细节，请查看我们不断丰富的设计文档集。要访问这些文档，只需加入 istio-team-drive-access@ 邮件列表即可。 加入之后，您就可以直接前往 文件夹（这里有很多技术文档）。
如果您想参与 Istio 项目，请考虑加入我们的 工作组 。
中文内容由 ServiceMesher 社区 维护，加入我们并参与进来吧！
